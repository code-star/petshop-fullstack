# Pet Shop App

![Screenrecording of the app](client/petshop-app.gif)

## Description

This is a pet shop app that allows :
- admin users to add pets
- customer users to adopt pets
- admin users and customer users to see the list of pets

## Technologies
- React
- Redux Query
- Typescript
- Material UI
